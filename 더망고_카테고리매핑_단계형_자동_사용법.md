# 더망고 카테고리매핑 단계형 자동 매크로 사용법

## 무엇을 하나
**단계형 카테고리매핑**(`admin_category_new.php`, 강의 2‑10)을 자동 처리합니다.
선택한 사이트(기본 **ABC마트**, THE MANGO 제외)의 카테고리 트리에서 **모든 최소단위(리프) 카테고리**를 찾아, 각 리프의 `카테고리 설정` 팝업(`admin_category_set.php`)을 순차로 열어:

**AI 자동매핑 → 11개 오픈마켓 카테고리를 규칙으로 재선택 → 고시 상품군/참조 처리 → 저장(fetch‑POST)** 을 반복합니다.

팝업 내부의 매핑 로직은 기존 **카테고리매핑(수동/자동) 매크로와 동일**합니다. 딱 한 가지가 다릅니다 ↓

## 수동/자동 매크로와 다른 점
- **11번가 = 국내 카테고리로 매핑.** ABC마트는 국내 사업자로 진행하는 사이트이므로(강의 2‑10), 11번가는 `해외` 카테고리를 피하고 **국내** 카테고리를 우선 선택합니다. (독일자라용 `자동` 매크로는 반대로 해외 우선.)
- 대상이 **필터가 아니라 사이트 카테고리 트리의 리프**입니다. 리프 열거는 사이트의 트리 데이터를 `admin_category_get_load.php`로 읽어(페이지 이동 없이) 수행합니다. ABC마트 기준 리프 약 122개.

## 자동으로 지켜지는 규칙 (강의 2‑10 기준)
- **금지어 회피(전 마켓)**: 어린이·유아·아동·도서·e쿠폰·모바일·렌탈·배달음식·출산·육아·임산부·위생용품·의료기기·의약품
- **옥션·G마켓**: `브랜드` 포함 카테고리 선택 금지 → 비‑브랜드로 대체
- **11번가**: `디자이너/biz` 금지 + **국내 카테고리 우선**
- **성별 일치**: 카테고리 경로의 남성(MEN)/여성(WOMEN)과 반대 성별 카테고리 배제 — 자동매핑이 성별을 틀리게 추천하면 교정
- 100% 일치는 불가하므로 **"비슷하면 OK"** 원칙으로 가장 근접한 카테고리 선택

## 동작 원리 (요약)
1. `admin_category_new.php`에서 사이트를 고르고 **리프 탐색 & 시작**을 누르면, 트리를 fetch‑DFS해 모든 리프(18자리 카테고리코드 + 경로명)를 큐로 만듭니다. 성별은 상위노드(MEN/WOMEN)에서 따와 경로명에 반영합니다.
2. 큐를 `localStorage`에 저장하고, 리프마다 `admin_category_set.php?category_id=…&ps_uid=…&tm=` 로 이동해 매핑·저장합니다(새로고침/이동을 넘어 진행이 이어짐).
3. 저장은 팝업/새로고침 없이 `admin_category_ok.php`로 **fetch‑POST**(다른 설정은 그대로 보존).
4. 완료되면 트리 페이지로 복귀하고, 결과 로그는 콘솔(F12)에 남습니다.

## 설치
1. Tampermonkey에서 새 스크립트 생성 → `더망고_카테고리매핑_단계형_자동.user.js` 내용 붙여넣기 → 저장.
2. 더망고 관리자에 **로그인**되어 있어야 합니다.
3. (독일자라용 `자동` 매크로와 함께 설치해도 됩니다. 서로 다른 `localStorage` 키를 써서 충돌하지 않습니다.)

## 사용
1. `정책적용 > 카테고리 관리 및 정책적용`(`admin_category_new.php`)으로 이동.
2. 우측 상단 초록색 패널:
   - **사이트**: 처리할 사이트 선택(기본 ABC마트, THE MANGO는 목록에서 제외됨).
   - **테스트(저장 안 함)**: 켜면 매핑을 계산만 하고 저장하지 않습니다(콘솔 F12에 결과 로그). **처음엔 켠 채로 몇 개 돌려 확인**을 권장.
   - **앞에서 N개만**: 앞 N개 리프만 처리(0=전체). 처음엔 3 정도로 시험.
3. **리프 탐색 & 시작** 클릭 → 리프 열거(약 10~20초) 후 확인창에서 진행. 이후 카테고리설정 페이지를 자동으로 순차 이동하며 처리합니다.
4. 중단하려면 **정지**(현재 항목까지 마치고 멈춤). 진행 상황은 패널과 콘솔에 표시됩니다.

## 검증 상태 (2026‑07‑13)
- **리프 열거**: ABC마트 122개 리프 정상 열거 확인(경로명·성별 정규화 포함).
- **매핑 팝업**: `admin_category_set.php`가 수동/자동 매크로와 동일 패널임을 확인(11개 마켓 셀렉트·AI자동매핑·`form_check`·`market_category`→`admin_category_ok.php`·`ps_uid`·`tm=''`).
- **저장 검증**: 자동매핑 후 `form_check` 통과("") 확인. 고시 상품군은 자동매핑이 11번가·롯데ON 포함 자동으로 채움. 저장 로직은 실사용 검증된 자동 매크로와 동일.
- **권장**: 실제 저장 전, **테스트 모드 + 앞 3개**로 한 번 돌려 콘솔 로그(특히 11번가 국내 여부·성별)를 확인한 뒤 전체 실행하세요.

## 주의
- 진행 중 페이지가 계속 이동합니다. 뜨는 팝업/스크래퍼 창은 건드리지 마세요.
- 엑셀·다른 매크로와 마찬가지로, 예외/오류/사이트 UI 변경 시에는 콘솔 로그를 확인하고 조정이 필요합니다.
