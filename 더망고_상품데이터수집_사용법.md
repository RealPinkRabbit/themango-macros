# 더망고 상품수집 자동 반복 (엑셀 연동) — 사용법 v3

`더망고_자동수집.user.js` 는 **엑셀(.xlsx)에서 필터/URL을 직접 읽어** 더망고 신규상품 수집을 순차 자동 등록하는 유저스크립트입니다. 데이터가 하드코딩되어 있지 않으므로, **엑셀만 고치면** 됩니다(코드 수정·재생성 불필요).

## 작동 개요
행마다 자동으로: **URL 검색(페이지 리로드) → 결과 로딩 대기 → 검색된 상품 모두저장 → 필터명 입력 → 저장상품수 3 → 저장하기 → 로그 "신규상품의 저장이 완료되었습니다" 대기 → 다음 행**.
검색이 페이지를 리로드하므로, 진행 단계를 브라우저에 저장해 새로고침을 넘어 이어갑니다. 팝업은 건드리지 않습니다.

## 설치 (Tampermonkey)
1. Chrome에 **Tampermonkey** 설치.
2. Tampermonkey → **새 스크립트 만들기** → 편집기 내용을 지우고 `더망고_자동수집.user.js` 내용을 **전체 붙여넣기** → **Ctrl+S**.
   - 이 스크립트는 엑셀 파싱을 위해 SheetJS 라이브러리를 `@require`로 불러옵니다(최초 1회 인터넷 필요, 이후 캐시).

## 엑셀 준비 (중요)
각 탭이 아래 열 구조를 따라야 합니다(현재 파일과 동일).
- **H3 = 사업자명** ← 반드시 실제 값. 예: ABC마트 탭 `솔데글KR`, 독일자라 탭 `솔데글GL`.
- **I3 = 수집사이트** (예: `ABC마트`, `독일자라`)
- **B열(3행부터) = 수집카테고리**, **D열 = URL**
- 필터이름은 `사업자명-수집사이트-카테고리` 로 자동 생성됩니다(엑셀 C열 수식과 동일).
- `원본` 이 들어간 시트(예시)는 자동 제외됩니다.

> 독일자라 탭 H3가 아직 `2번사업자명` 이면 **`솔데글GL` 로 바꾼 뒤** 저장하세요. 안 그러면 필터명이 `2번사업자명-...` 로 만들어집니다.

## 사용
1. 더망고 관리자 **로그인** + 팝업/확장 허용(기존 세팅).
2. `상품데이터수집 → 신규상품 수집`(getGoods.php)으로 이동 → 우측 상단 **빨간 패널**.
3. **"엑셀 불러오기(.xlsx)"** 클릭 → 파일 선택. 불러오면 탭별 `사업자명 / 행수` 확인 알림이 뜹니다.
4. 아래에 **사이트별 시작 버튼**(예: `ABC마트 시작 (92)`, `독일자라 시작 (116)`)이 생깁니다. 원하는 것을 클릭.
5. 진행 상황이 패널에 표시됩니다. 탭은 켜둔 채로 두세요.

### 버튼
- **기존 필터 건너뛰기**(기본 켜짐): 이미 있는 필터는 건너뜁니다.
- **정지**: 현재 행 완료 후 멈춤.
- **이어서**: 정지/오류 지점부터 재개(기존 필터 재확인 후 진행).

## 데이터 갱신 / 확장
- 카테고리·URL·사업자명이 바뀌면 **엑셀만 수정 후 다시 "엑셀 불러오기"**. (불러온 데이터는 브라우저에 저장되어, 바꾸기 전까지 유지됩니다.)
- 새 소싱사이트도 **같은 열 구조로 탭을 추가**하면 자동 인식됩니다. 단, 더망고 사이트 탭 코드 매핑이 필요합니다 — 현재 지원: `ABC마트(a_rt)`, `독일자라(zara_de)`, `자라KR(zara_kr)`, `무신사(musinsa)`, `eBay(ebay)`. 이 외 사이트를 쓰려면 스크립트의 `LABEL_TO_CODE`/도메인 매핑에 한 줄 추가가 필요합니다(요청 주세요).

## 참고 / 한계
- 한 행당 대략 40~70초. 오류로 멈추면 알림 후 정지 — 수동 확인 뒤 "이어서".
- "기존 필터 건너뛰기"는 목록에 보이는 필터 기준 best-effort(페이지네이션 시 일부 중복 가능).
- 더망고 화면(버튼/입력창)이 크게 바뀌면 셀렉터 수정이 필요할 수 있습니다.
