# 더망고 신규상품수집 — 저장상품수 일괄변경 (사용법)

`더망고_저장상품수_일괄변경.user.js` 는 신규상품수집 목록의 **모든 필터의 "저장상품수(limit_count)"를 지정값(기본 100)으로 일괄 변경**하는 유저스크립트입니다. (필터 생성 시 3개로 만든 뒤, 본수집 전에 100개(또는 200개)로 올리는 단계를 자동화)

## 작동 방식 (빠르고 안전)
- **팝업·새로고침·페이지 이동 없이** 순수 fetch로 처리합니다.
- 각 필터의 수정폼을 읽어 **저장상품수만 바꾸고 나머지 설정(카테고리 매핑·번역·정책 등)은 그대로** POST 저장합니다.
- 정적 파싱 값이 실제 렌더 값과 100% 동일함을 확인했고, 저장 후 다른 필드가 보존됨을 검증했습니다.

## 설치 (Tampermonkey)
1. Tampermonkey → **새 스크립트 만들기**
2. `더망고_저장상품수_일괄변경.user.js` 내용을 **전체 붙여넣기** → **Ctrl+S**

## 사용
1. 더망고 관리자 로그인 상태에서 `상품데이터수집 → 신규상품 수집`(getGoodsCategory.php)으로 이동.
2. 우측 상단 **파란 패널**에서:
   - **저장상품수**: 목표값 입력(기본 100. 3만개 업그레이드 시 200).
   - **대상**: 전체 / ABCmart / Zara.com/de 중 선택.
3. **시작** 클릭 → 전체 필터를 자동으로 열거하고 하나씩 변경합니다. 진행 상황이 패널에 표시됩니다.
4. **정지** 버튼으로 중단 가능.

## 특징
- **이미 목표값(예: 100)인 필터는 자동 건너뜀** → 재실행해도 안전(중단 후 다시 시작하면 남은 것만 처리).
- 실패한 필터가 있으면 패널에 **실패 ID**가 표시됩니다. 다시 "시작"하면 재시도됩니다.
- 200개 필터 기준 대략 2~4분.

## 한계 / 유지보수
- 더망고 화면·내부 함수가 바뀌면 수정이 필요할 수 있습니다. 현재 의존 요소:
  - 목록 페이지네이션: `getGoodsCategory.php?ft_num=100&pg=N` (파라미터 `pg`)
  - 수정폼 URL: `modify_filter(필터ID)` 의 window.open 대상 (`admin_group_modify.php`, `/mall/admin/`)
  - 저장 endpoint: 수정폼 action = `admin_etc_ok.php` (`/mall/admin/`), 저장상품수 필드 = `limit_count`
