# 더망고 환율정보 일괄갱신 매크로 사용법

`admin_config.php`의 **"환율정보 등록"** 12개 통화(USD, EUR, CNY, JPY, GBP, AUD, SGD, CAD, HKD, TRY, NZD, PLN)를 **시장환율 + 마진%(기본 4%)** 로 자동 갱신하고, 환율 섹션 **설정저장**까지 실행합니다.

## 동작 원리
- 무료 환율 API(키 불필요, CORS 허용)에서 USD 기준 환율을 받아, 각 통화의 **원화값 = (1USD 원화) ÷ (1USD당 해당통화)** 로 계산합니다.
- 계산된 시장환율에 **마진%**(패널에서 조정, 기본 4)를 곱해 입력값을 만듭니다. `입력값 = 시장환율 × (1 + 마진/100)`.
- **반올림**: 값이 100 이상이면 원단위 정수(예: USD 1,559), 100 미만이면 소수 2자리(예: JPY 9.64, TRY 33.16) — 사이트 기존 표기와 동일한 방식입니다.
- 저장은 환율 섹션의 `설정저장`(= `config_save(1)`)이 하는 것과 동일하게 **`menu_id=1` 로 `morning` 폼을 `admin_config_ok.php` 에 POST** 합니다. **menu_id=1(환율 섹션)만 저장**되므로 회사정보·API키·배송사 등 다른 설정은 변경되지 않습니다.
- 환율 소스는 3곳을 순서대로 시도합니다: currency-api(jsdelivr) → currency-api(pages.dev) → open.er-api.com. 앞 소스가 실패하면 자동으로 다음으로 넘어갑니다.

## 설치
1. Chrome에 Tampermonkey 확장 설치.
2. Tampermonkey 대시보드 → 새 스크립트 → `더망고_환율정보_일괄갱신.user.js` 내용 전체를 붙여넣고 저장(Ctrl+S).
3. 더망고 관리자에 **로그인**한 상태로 `https://tmg4682.mycafe24.com/mall/admin/admin_config.php` 를 열고 **"환율정보 등록"** 화면으로 이동.

## 사용
페이지 우측 상단 **"환율정보 일괄갱신"** 패널:

1. **마진 %**: 시장환율에 더할 비율(기본 4). 필요하면 값을 바꿉니다.
2. **① 환율 불러오기·미리보기**: 환율을 받아 표에 `통화 / 시장환율 / +마진% / 반영값 / 현재값` 을 보여줍니다. **입력칸은 아직 건드리지 않습니다.** (마진을 바꿨으면 이 버튼을 다시 눌러 재계산)
3. 확인 후:
   - **② 값만 채우기**: 12개 입력칸을 계산값으로 채우기만 합니다(**저장 안 함**). 직접 `설정저장`을 누르거나, 새로고침하면 원복됩니다.
   - **② 채우고 저장**: 입력칸을 채운 뒤 **확인창**에 변경 내역을 보여주고, `확인`을 누르면 설정저장까지 실행합니다. `취소`하면 값만 남고 저장은 안 됩니다.
4. 하단 상태줄/로그로 소스·기준일·성공 개수·오류를 확인합니다.

패널 헤더를 드래그해 위치를 옮길 수 있고, `_` 로 접을 수 있습니다.

## 주의 / 검증 메모
- **저장은 환율 섹션(menu_id=1)만** 반영됩니다. 다른 섹션 설정은 그대로 유지됩니다.
- 미리보기·값만채우기는 **비파괴**(저장 전까지 새로고침으로 원복). 실제 검증에서 12개 통화가 정확히 계산·입력됨을 확인했습니다(JPY·TRY는 소수 2자리, 그 외 정수+콤마, 미설정이던 HKD도 채워짐).
- 환율은 실시간 시세이며 API 기준일이 하루 전일 수 있습니다(로그에 기준일 표시). 강의 방식대로 **마진 4%** 가 기본값이며 패널에서 조정 가능합니다.
- 만약 세 소스가 모두 실패하면(네트워크/차단) 로그에 표기되고 채우기가 진행되지 않습니다. 잠시 후 다시 시도하세요.
- 입력칸을 못 찾으면(다른 설정 탭) 패널이 안내 메시지를 표시합니다. "환율정보 등록" 화면에서 실행하세요.
