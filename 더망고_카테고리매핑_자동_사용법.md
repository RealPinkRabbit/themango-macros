# 더망고 카테고리매핑 자동 매크로 사용법

## 무엇을 하나
검색필터 세부설정(`admin_category_set.php`)의 **(오픈)마켓 카테고리 매핑**을 필터별로 자동 처리합니다.
각 필터에 대해: **설정 열기 → AI 자동매핑 실행 → 11개 마켓 카테고리를 규칙으로 재선택 → 저장(fetch‑POST) → 다음 필터**.
대상은 **Zara(독일자라) 필터 전체**입니다. 상태 무관 — 아직 매핑 안 한 "설정하기" 필터뿐 아니라 **이미 완료된 "설정수정" 필터도 다시 매핑**합니다. ABC마트는 방식이 달라 제외합니다.

## 자동으로 지켜지는 규칙 (강의 2‑14 + 문서 기준)
- **금지어 회피(전 마켓)**: 어린이·유아·아동·도서·e쿠폰·모바일쿠폰·렌탈·배달음식·출산·육아·임산부·위생용품·의료기기·의약품
- **옥션·G마켓**: `브랜드`로 시작하는 카테고리 선택 금지 → 비‑브랜드 카테고리로 대체
- **11번가**: `해외` 카테고리 우선 선택 + 상품고시 상품군을 **의류**로 설정(저장 검증 통과에 필요)
- **성별 일치**: 필터이름의 남성/여성과 반대 성별 카테고리 배제(자동매핑이 성별을 틀리는 경우 교정)
- 100% 일치는 불가하므로 **"비슷하면 OK"** 원칙으로 가장 근접한 카테고리를 고릅니다.

## 동작 원리 (요약)
- 원시 카테고리 코드를 손으로 미리 채우지 않습니다. 대신 사이트의 **AI 자동매핑 추천 + 카테고리 검색**을 런타임에 규칙으로 걸러 고릅니다.
  1. 자동매핑 추천 5개 중 규칙을 통과하는 최적 후보가 있으면 그것을 선택
  2. 없으면(예: 옥션 추천이 전부 `브랜드`) 상품유형 키워드(예: "여성 슬랙스")로 **검색**해 통과 후보 선택
- 저장은 팝업/새로고침 없이 `admin_category_ok.php`로 **fetch‑POST**(다른 설정은 그대로 보존).
- `localStorage`로 새로고침/페이지 이동을 넘어 진행이 이어집니다.

## 설치
1. Tampermonkey에서 새 스크립트 생성 → `더망고_카테고리매핑_자동.user.js` 내용 붙여넣기 → 저장.
2. 더망고 관리자에 **로그인**되어 있어야 하고, 팝업 허용 상태를 권장합니다.

## 사용
1. `정책적용 > 상품수집조건(검색필터) 관리 및 정책적용` 페이지(`admin_group.php`)로 이동.
2. 우측 상단 보라색 패널:
   - **테스트(저장 안 함)**: 켜면 매핑을 계산만 하고 저장하지 않습니다(콘솔 F12에 결과 로그). **처음엔 켠 채로 몇 개 돌려 확인**을 권장.
   - **앞에서 N개만**: 앞 N개만 처리(0=전체). 처음엔 3 정도로 시험.
   - **대상 수집 & 시작**: 독일자라 필터(설정하기+설정수정 전체)를 모아 순차 처리 시작.
3. 진행 중 페이지가 계속 이동합니다. **스크래퍼 팝업창은 절대 건드리지 마세요.**
4. 완료 후 결과 로그는 브라우저 콘솔(F12)에 출력됩니다. `설정하기`가 `설정수정`으로 바뀌면 성공.

## 확인된 사실(검증됨)
- 설정 팝업 실제 주소: `admin_category_set.php?tm=F&ps_ftid={필터ID}` (필터ID는 목록의 `market_mapping_new('ID')`).
- 저장 폼 = `market_category`(POST → `admin_category_ok.php`), 저장 전 `form_check()`가 검증 문자열 반환 시 실패.
- **고시(notify) 처리**: 저장 검증(`form_check`)은 면제 외 마켓의 카테고리가 지정되면 상품군(notify_group_no)이 채워져 있어야 통과. 대부분은 자동설정되지만 **11번가·롯데ON은 자동설정되지 않아** 비면 "OO 고시정보를 선택하세요"로 실패함(구버전 대량 실패의 주원인). → 스크립트가 해당 마켓의 **상품군을 지정(의류 등)하고 "상품상세페이지 참조" 체크박스를 체크**해 해결(대표님 절차). 검증됨.
- 마켓 코드: 옥션=AUC20, G마켓=GMK20, 11번가=11ST, 스마트스토어=SMART, 쿠팡=COUP, 롯데ON=LTON, LF몰=LFMALL, 머스트잇=MUSTIT, 쇼피=SHOPEE, 큐텐JP=QOO10JP, 플레이오토=PLAYAUTO.
- AI 자동매핑/저장검증은 `alert()`로 렌더러를 멈추므로 스크립트가 `alert`을 무력화하고 진행.

## 주의
- **처음엔 반드시 테스트 모드 + 소수(3개)로** 결과를 콘솔에서 확인한 뒤 전체 실행하세요.
- 카테고리는 근사 매칭입니다. 특정 필터의 매핑이 마음에 안 들면 해당 필터만 수동으로 열어 조정하면 됩니다.
- 자동매핑 추천/검색 결과에 통과 후보가 전혀 없으면 해당 마켓은 자동매핑 원값을 유지(`(유지)`로 로그)합니다.
